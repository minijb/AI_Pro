## Text Formatting

This is **bold text** using double asterisks.
This is __bold text__ using double underscores.

This is *italic text* using single asterisks.
This is _italic text_ using single underscores.

This is ***bold and italic*** text.

This is ~~strikethrough~~ text.

This is ==highlighted text== (Obsidian-specific).

This is `inline code` text.

---

## Escaping Special Characters

\*This is not italic\*
\#This is not a heading
1\. This is not a list item
\[[This is not a wikilink\]]
\==This is not highlighted\==

---

## Paragraphs and Line Breaks

This is the first paragraph. It has multiple sentences in a single block.

This is the second paragraph. A blank line separates paragraphs.

This line has a line break  
right here (two trailing spaces before the newline).

This line uses an HTML break<br>to create a new line.

## Horizontal Rules

Three hyphens:

---

Three asterisks:

***

___

- - -
